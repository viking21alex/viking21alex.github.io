1. 物业、厂房和设备（续）

物业、厂房及设备主要核算的是归实地集团及其子公司所有的固定资产、租赁资产改良以及在建工程，固定资产主要是房屋建筑物和与日常经营相关的电子设备、办公设备、运输设备，租赁资产改良主要为桉树空间将租赁的房屋装修改造再出租所发生的装修改造成本，在建工程主要为用于自持的在建的三亚的华尔道夫酒店、在建遵义的希尔顿花园酒店、北京实地正在装修改造用于高端营销招待的一套房产以及北京东城得丰东巷4号品牌馆进行精装修。本集团房屋建筑物主要为实地地产的保利天悦房产、遵义实地的售楼中心大楼，恒逸设备用于办公的盈凯广场2103～2108单元房屋。我们已获取并检查房屋产权证明。

2020年全年，物业、厂房和设备外购人民币203,247千元，主要原因如下：各家子公司随着公司规模的扩大和电子设备更新的需要，购置了电脑、扫描仪、打印机等电子设备人民币2,794千元，其中广州实地2020年新购入电子设备人民币2,026千元,重庆实创、安阳太富、昆明中和实地分别新购入电子设备人民币351千元、人民币173千元和人民币161千元；另外购沙发、会议桌椅等办公设备人民币1,255千元，此外同年新增运输设备为人民币2,895千元，此部分新增为广州实地、广州丰实、太原和泰盛瑞、株洲实地、钜融机电5家公司购置的小汽车，新增额分别为人民币1,429千元、人民币642千元、人民币325千元、人民币269千元、人民币230千元。在建工程增加人民币194,992千元，其中遵义实地的希尔顿花园酒店增加人民币2,305千元，三亚希尔顿华尔道夫酒店增加建安工程、开发间接费共人民币186,573千元。当期折旧为人民币28,555千元，系2020年全年计提折旧所致。

2. 投资性房地产（续）

该集团的投资性房地产主要包括用于出租的商业、会所、幼儿园等。对于投资性房地产，集团遵照《国际会计准则第40号-投资性房地产》采用“公允价值模式”进行计量。每个资产负债表日的投资性房地产的公允价值均按独立的拥有专业资格的评估师仲量联行（JLL）进行评估确认。2020年12月31日投资性房地产余额较2019年末增加人民币54,500千元，主要原因如下：

  1. 在建投资性房地产持续建设导致投资性成本增加32,995千元所致。其中，广州常春藤项目4号地和2号地随着项目持续建设成本分别增加人民币3,905千元和12,997千元；青岛蔷薇国际Z1组团成本增加人民币5,385千元；遵义君兰汀岸项目Z1独立商业和Z2独立商业成本分别增加人民币4,761千元和人民币2,982千元；深圳剑兰郡项目成本增加人民币4,330千元。此外，广州蔷薇国际项目S4至S8商铺 2020年工程实际结算成本相比预提成本金额减少，导致成本减少人民币1,365千元。此外，武汉君兰汀岸项目Z1独立商业的36#和会所本年项目状态由在建转完工，本年已获取竣工备案证，项目状态转为完工。其余已完工投资性房地产本年成本无变化。

    （b）公允价值变动21,505千元主要是在建投资性物业的估值增加，随着工程进度推进导致建安成本增加，公允价值增加是因为在使用假设开发法评估方法下，估值随着成本增加其所含利润及管理费用等计入公允价值，具体评估分析详见Consol_SRM+Appendix+II_OAR_PL Note 6. 公允价值变动损益

    对于以上各类投资性房地产项目的公允价值，公司主要以评估师按收益法、假设开发法评估的结果在财务报表中反映。空地或在建投资性房地产主要用假设开发法，即项目的公允价值通过估计该项目开发完毕后实现的销售收入和扣除尚需发生的开发成本、销售费用、销售税金、开发商利润等各项项目后的剩余价值来计算。对完工项目使用收益法（租赁期法）进行评估，即计算开发完毕后实现的销售收入；租赁期法考虑现有租金和租赁期后潜在租金收益，并采用一个合适的资本化率将租金资本化得出物业的公允价值；收益法主要适用于完工投资性房地产，通过将整租赁期内及租赁期外的租金折算的现值确认。根据企业会计准则第39号-公允价值计量，假开法和收益法均属于可使用的估值技术，安永TAS部门对估值方法表示认可。我们引入TAS部门的专家对广州常春藤2号地、中山百合家园、深圳剑兰郡20年公允价值变动较大的投资性物业的评估方法和主要评估参数及评估结论的合理性进行复核，并出具分析结论，未见异常。

    对于收益法，其主要的评估参数是评估租约期内价值的现有租赁合同的租金、租赁期和租赁期内回报率计算得出的。租赁期外价值是基于市场租金、市场回报率及物业的剩余可使用年限的得出的，市场租金主要是参考周边可比类型投资性房地产的租金水平以及整体市场行情，对现有租金进行适当的调整。TAS部门主要对租赁期回报率、市场租金、市场回报率、楼层调整系数、剩余租赁期限进行复核并出具意见。我们主要对租约内租金及租赁期进行复核；租约内租金通过检查租赁期内的账面入账租金确定；租赁期主要是根据租赁合同期确定，未发现异常。

    假设开发法房地产项目的公允价值通过估计该项目开发完毕后实现的销售收入和扣除尚需发生的开发成本、销售费用、销售税金、开发商利润等各项项目后的剩余价值来计算。TAS复核对假设开发法计算中使用的开发完毕后实现的销售单价、开发商利润参数进行复核并出具意见，我们通过检查项目的预算成本表，复核项目的续建成本及其他销售费用、销售税金、利润率等项目明细，未见异常。

7.4 计提土地增值税确认的递延所得税资产于2020年12月31日和2019年12月31日的余额分别为人民币370,937千元和人民币129,703千元。该项递延所得税资产形成的原因为已结利项目公司清算口径计提的土地增值税与按照预缴口径税前可扣除的土地增值税差异所致，税法不允许本年抵扣超过实际预交土增税的税金及附加金额，根据税法要求在实际缴纳的时候允许抵扣土增税产生的税金及附加形成了暂时性差异，因此确认的递延所得税资产。广东誉丰、广州丰实在2020年1-12月年均有部分项目房屋交付结利使得相应人民币749,256千元、人民币270,949千元的土地增值税结转至当期损益，根据税法规定，这部分允许在土增税实际清算缴纳时予以税前扣除，故与本期预缴形成暂时性差异人民币659,545千元、人民币229,862千元，确认的递延所得税资产较上年增加人民币164,886千元、人民币57,465千元。

7.6 该余额为会计上投资性房地产评估增值和税法允许的投资性房地产折旧形成的暂时性差异。投资性房地产以公允价值模式计量，根据投资性房地产评估报告计算房产增值，相应确认递延所得税负债；以公允价值模式计量的投资性房地产，是以公允价值的变动计算相应的损益，不计提折旧。《企业所得税法》未对以公允价值模式计量的投资性房地产做特殊规定，因此其税务处理应按税法对资产的一般性规定进行处理。在实际征管中，凡符合税法规定的固定资产、无形资产，应按税法规定计算折旧、摊销，并准予扣除。会计与税法不一致的，应按照税法规定调整。会计上该投资性房地产的账面价值为其公允价值，计税基础为取得成本扣除按照税法规定允许税前扣除的折旧额后的金额，该部分应纳税差异形成递延所得税负债。我们对2020计入的递延所得税费用进行测算， 2020年投资性房地产的累计公允价值变动收益及折旧影响为人民币129,494千元，确认递延所得税负债人民币32,373千元。

案例4《会议纪要》–关于合并范围内股权转让相关递延所得税的确认​

2022年第四次会议

【背景或案例】上市公司甲持有乙公司100%的股权，甲公司对乙公司长期股权投资账面价值为2亿元。2021年12月，甲公司以1.4亿元的价格将其持有的乙公司股权全部转让给全资子公司丙，并将由此产生的股权转让亏损0.6亿元在计算当期所得税时予以扣除（甲公司适用税率为25%），从而少交当期所得税1500万元。甲公司在其 2021 年度合并报表中是否需要确认1500万元递延所得税负债？

会议分析，多数与会代表认为，该问题相关的税会差异属于企业合并报表层面的“外部差异”，应当从合并报表层面将子公司的股权投资作为整体来看，案例所述股权转让交易产生了长期股权投资计税基础的变化，即由甲公司的长期股权投资计税基础2亿元变成了1.4亿元，其与长期股权投资的账面价值（按照权益法核算的结果）一般不会导致在合并报表层面产生暂时性差异；即使产生一定金额的暂时性差异，也很可能符合《企业会计准则第18号——所得税》（以下简称所得税准则）第十二条、第十四条中豁免确认递延所得税的条件而不确认递延所得税。部分与会代表认为，企业还需要进一步结合企业持有子公司目的来判断是否符合所得税准则第十二条中豁免确认递延所得税负债的条件，如果不是以出售为目的，一般符合递延所得税负债豁免确认条件；如果是以出售为目的，一般不符合递延所得税负债豁免确认条件，往往需要确认递延所得税负债。

会议初步意见，该问题具有一定的普遍性和代表性，会议讨论形成倾向性意见，企业应当按照所得税准则第十二条、第十四条等有关规定对是否符合豁免确认递延所得税的条件从而对是否确认递延所得税进行判断。

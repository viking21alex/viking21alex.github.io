案例3《会议纪要》–关于视频网站付费收入的确认​

2021年第二次会议

【背景或案例】某公司运营儿童早教视频网站平台，部分视频只需注册就可以免费观看，另有部分视频需要付费观看。付费有两种模式：一是购买会员资格，在会员期限内可观看向会员开放的多个视频，二是单独付费购买特定视频，可无限期反复观看。两种模式下，客户均随时下载相关视频到本地APP中离线观看。对于会员模式，实务中一般采取在会员期限内分摊确认收入。对于单独付费模式，企业应一次性确认收入还是分摊确认收入？若采用分摊确认收入，摊销期限应如何估计？

会议分析：该问题具有普遍性和代表性，与会代表认为应当结合业务模式和具体情形进行会计处理。对于会员购买视频后可无限期反复观看并可随时下载，且公司有义务维持平台网站的正常运营以维持该视频资源的情形，企业应在一段时间内分摊确认收入，具体分摊期限应根据具体视频类型、客户粘性等不同有所差异。对于企业将客户已购买的视频已交付且没有后续义务的情形，如下载的视频可以离线观看且不支持随时再次下载的，企业应一次性确认收入。

会议初步意见：虽然该业务具有普遍性和代表性，但现行《企业会计准则第14号——收入》已有明确规定，企业应按照该准则，结合具体业务安排进行具体分析，并相应进行会计处理。拟通过会议纪要形式统一理解认识。

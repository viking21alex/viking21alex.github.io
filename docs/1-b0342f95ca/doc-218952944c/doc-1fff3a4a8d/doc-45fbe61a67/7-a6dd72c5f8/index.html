案例7《会议纪要》–关于属于某段时间内履行的知识产权许可按照客户实际销售或使用情况收费的收入确认​

2023年第一次会议

【背景或案例】情形一：A公司是一家游戏开发公司，其开发的某款游戏在国内国外均有运营发行，国内为A公司自主运营，国外从2022年起授权B公司独家运行。A公司负责后续的更新升级服务。由于该游戏的设计特点是根据玩家的实时反馈进行频繁的更新升级（每周由B公司下达指令给A公司进行更新），才能保证对玩家的持续吸引力，因此A公司的授权许可需与后续升级结合在一起，整体作为一项履约义务。假设公司判断该履约义务中知识产权许可占主导地位，由于公司后续活动对该许可的使用有重大影响，因此该许可属于一段时间内履行的履约义务。A公司根据B公司运营游戏取得玩家充值的40%分成。

情形二：C公司是一家汽车语音系统提供商，其客户是整车厂商D。根据C与D的协议约定，由C提供一套车载语音系统安装到D的某一款新车型中供购车人使用，通过语音搜索行车路线、餐饮娱乐等信息。车载语音系统的知识产权由C公司所有，授权D公司使用，使用期间包括该款车型的生产销售期间及停止销售之后的3年（根据以往的销售经验估计该车型销售期为5年，因此预计总的授权使用期间为8年）。C公司承诺该系统软件的功能可以在上述期间内持续有效运行，为此C公司需要在整个授权期内持续更新地址、路线等信息，因此该授权许可收入属于一段时间内履行的履约义务。C公司每月根据安装了该语音系统的汽车销售的数量，按10元/台的价格收取特许权使用费。

对于上述某段时间内履行义务，并约定按客户实际销售或使用情况收取特许权使用费的知识产权许可，如何判断履约进度与销售或使用行为发生二者孰晚，并在孰晚的时点确认收入？

会议分析，对于情形一，多数与会代表认为，游戏软件授权使用并持续更新的履约行为，与游戏玩家的持续充值是有密切关联的，可通过定性分析认为履约进度与销售发生的时间大致相当，从而按照充值分成确认收入。个别与会代表认为，可以从成本的角度考虑履约进度，游戏软件授权使用的第一天已有较高比例的履约进度，从而认为充值时间较晚，可以按充值确认收入。个别与会代表认为，A公司是否能够确认收入取决于B公司是否已经向玩家履行履约义务，玩家充值并不代表B公司已经履行履约义务，除非玩家充值与实际消费购买道具或消耗道具之间，不存在重大时间差，才能按充值分成确认收入。

对于情形二，多数与会代表认为，语音系统授权使用并持续更新的履约行为与汽车销售的发生没有必然关联，且根据背景描述，授权使用的进度很可能晚于汽车销售的时点，因此不能按照实际销售收费的金额确认收入。在此前提下，每一笔收费发生时，应按照履约进度进行分摊确认收入，具体计算方面存在两种观点，部分与会代表认为，取得每笔收费时，应累计计算销售发生前已履约（已授权使用）的期间，按累计的进度分摊确认收入；部分与会代表认为，取得每笔收费时，在剩余的授权使用期内按履约进度确认收入。个别与会代表认为，可以将该业务视为多个小合同，每个小合同对应一个授权即一个单独的履约义务，其对价是固定的10元/台，在取得该项收费时，在剩余的授权使用期内分摊确认收入。个别与会代表认为，美国会计准则下的相关实务案例中还有一种可接受的做法，即企业对整个销售周期内的汽车销售情况作了最佳估计，并以此估计合同总对价，在整个授权期内按履约进度分摊确认收入，但前提是确保每期分摊确认收入的金额不高于当期实际收到的使用费。

会议初步意见，该问题具有一定的普遍性和代表性，会议讨论初步形成倾向性意见，对于某段时间内履行的知识产权许可，需按照企业会计准则的有关规定考虑计量履约进度的基础，判断履约进度与客户后续销售或使用行为的发生孰晚，并按照现行准则规定进行会计处理。

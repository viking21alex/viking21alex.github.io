案例12《会议纪要》–时段履约义务的判断

案例12《会议纪要》–时段履约义务的判断​

2024年第三次会议

【背景或案例】A公司为一家提供系统集成设备的科技创新企业，单个项目具有金额大、实施周期长的特点。A公司生产销售过程分为两个阶段：

1.设备组件生产阶段。A公司自身不生产系统集成设备中的相关组件，而是委托供应商生产，生产完工后由供应商直接发运至客户项目现场。各组件均为标准化通用件，可以在A公司不同项目间通用。

2.现场安装和调试阶段。各组件发运至客户项目现场后，A公司于项目现场进行分批安装，并嵌入标准化软件平台，最终向客户交付系统集成设备。A公司还负责系统集成设备的整体调试，以及与客户其他系统（非A公司提供）的联合调试。

由于A公司需提供重大的服务以将合同中承诺的组件、安装和调试整合成合同约定的系统集成设备转让给客户，上述商品或服务构成单项履约义务。问题：对于该单项履约义务，在判断“客户是否能够控制企业履约过程中在建的商品”时，A公司的组件生产阶段是否属于履约过程？

会议分析，多数与会代表认为，基于本案例的背景情况，A公司生产的组件均为标准化通用件，并非按客户需求定制化生产的专用件，这与制造业企业生产存货的性质类似，因此在组件生产阶段A公司尚未开始向客户履约，组件生产阶段不属于履约过程。部分与会代表进一步指出，在此前提下，A公司不能简单认定在现场安装和调试阶段客户能够控制履约过程（即组件发运至现场后的安装和调试过程）中在建的商品，而应当结合合同约定和交易实质具体分析判断。个别与会代表认为，A公司生产组件已经构成向客户履约，组件生产阶段属于履约过程。

会议初步意见，该问题具有一定的代表性，会议讨论初步形成倾向性意见，本案例中的组件生产阶段不属于履约过程；企业应当根据会计准则规定，结合合同约定和交易实质，判断客户是否能够控制企业履约过程中在建的商品。

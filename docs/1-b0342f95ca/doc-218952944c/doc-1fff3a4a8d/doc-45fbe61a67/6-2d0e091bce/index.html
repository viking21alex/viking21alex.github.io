案例6《会议纪要》–关于控制权在某一时点转移的业务，亏损合同如何进行会计处理

案例6《会议纪要》–关于控制权在某一时点转移的业务，亏损合同如何进行会计处理​

2023年第一次会议

【背景或案例】《企业会计准则解释第 15 号》关于亏损合同的判断，明确了履行合同的成本包括：（1）履行合同的增量成本；（2）与履行合同直接相关的其他成本的分摊金额。其中，履行合同的增量成本包括直接人工、直接材料等；与履行合同直接相关的其他成本的分摊金额包括用于履行合同的固定资产的折旧费用分摊金额等。当履行合同的成本超过预期经济利益时，根据或有事项准则，先对标的资产进行减值测试并按规定确认减值损失，如预计亏损超过计提的减值损失，应将超过部分确认为预计负债。对于控制权在某一时点转移的业务，计提的预计负债计入“主营业务成本”还是“营业外支出”？

会议分析，与会代表一致认为，财政部发布的《收入准则应用案例——亏损合同案例》中针对在某一时段内履行的履约义务，明确了当待执行合同变为亏损合同的，计提的预计负债计入主营业务成本，对于控制权在某一时点转移的业务，预计亏损同样与履行客户合同相关，也应计入主营业务成本。此外，部分与会代表还建议，对于计提的存货跌价准备，在出售存货时也应同时将存货跌价准备转入主营业务成本，以保持会计处理的一致性。

会议初步意见，该问题具有一定的普遍性和代表性。会议讨论初步形成倾向性意见，在履行客户合同时，无论控制权在某一时点转移还是在一段时间内转移，当待执行合同变为亏损合同的，计提的预计负债应计入主营业务成本。下一步拟研究通过实施问答或修订指南等适当方式指导实务。

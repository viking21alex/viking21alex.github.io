案例15《会议纪要》–关于新能源开发公司出售光伏电站项目附带保证条款的会计处理问题​

2025年第二次会议

【背景或案例】新能源开发公司A的主营业务为投资开发并出售户用光伏电站项目，通过转让项目公司股权或出售电站资产的方式获取经济利益。出售价格通常以光伏电站的全生命周期（一般为25年）的预计净现金流量现值为基础确定。新能源开发公司A的子公司B负责户用光伏电站全生命周期的运维工作。新能源开发公司A及运维公司B共同向资产收购方C及项目公司D出具相关保证。实务中常见的保证情形如下：

情形一：全生命周期保障发电量。为保证光伏电站质量，收购方要求依据电站所在区域等信息计算电站的预计发电小时数，新能源开发公司A及运维公司B共同承诺电站在全生命周期达到预计的发电小时数。待光伏电站交易后，资产收购方C和项目公司D按月对电站发电量进行监测，发电量低于保证发电小时数（以下简称保发小时数）的，新能源开发公司A或运维公司B支付发电量保证补偿款，计算公式为：发电量保证补偿款=（保发小时数－实发小时数）×脱硫煤电价。同时，合同中还约定新能源开发公司A或运维公司B可以获得超发电量的分成收益。

情形二：在情形一的基础上，进一步增加对全生命周期电价或收益率的保证。

（1）电价保障条款。当光伏电站后续实际电价未达到合同约定的单瓦理论电价（通常为合同签署时项目所属地电网的电费结算单价）时，新能源开发公司 A 需承担电价差额补偿责任，计算公式为：电价差额补偿款=项目当年保发小时数×（单瓦理论电价－当年项目公司电力实际结算均价）。

（2）收益率保障条款。根据双方议定的投资收益率测算交易价格，新能源开发公司A对项目投资收益作出如下保证：每年进行收益率复核，若不满足投资收益率，则由新能源开发公司A补偿资产收购方C损失直至满足投资收益率。

问题：上述两种情形下，新能源开发公司A合并财务报表层面能否在出售电站时确认电站出售收入？

会议分析，对于情形一，多数与会代表认为，新能源开发公司A出售项目公司D后，合并报表中的电站资产是否应当出表，取决于新能源开发公司A及运维公司B共同向资产收购方C及项目公司D出具的相关保证的本质。通常意义上的保证类质保只是针对产品本身的质量保证，并不涉及对其他不确定因素（如光伏电站发电通常受阳光等自然条件、电网接入能力有限或运行不稳定等电网因素影响）进行保证。如果“全生命周期保障发电量”的保证条款本质上不仅涉及产品质量本身的保证，还涉及对其他不确定因素的保证，则该保证并非通常意义上的保证类质保。本案例中相关保证承诺覆盖电站全生命周期，往往表明出售电站的主要风险并未转移，结合合并财务报表准则有关控制权因素综合分析后，通常应认为电站控制权尚未转移。个别与会代表认为，如果保证条款属于合理的产品质量保证范畴，则为保证类质保，即使提供全生命周期保证，也有可能表明控制权已经转移。部分与会代表还提出，实务中该类业务的保证类质保一般为3-5年的保证期限，如果提供全生命周期质量保证且负责全生命周期的运维，通常表明企业提供保证的风险很高，结合合并财务报表准则有关控制权因素分析后，通常应认为电站控制权尚未转移。

对于情形二，与会代表一致认为，新能源开发公司A在出售项目公司D给收购方C的约定中，在保证全生命周期发电量的基础上，同时保证了全生命周期的电价或者投资收益率，通常表明新能源开发公司A对于项目公司D仍然继续承担上网电量、价格等方面的重大经营风险，项目公司D的控制权尚未转移。

会议初步意见，该问题具有一定的代表性，对于情形一，会议讨论初步形成倾向性意见，基于本案例的背景，需要判断企业提供全生命周期发电的保证条款的实质，如果涉及对合理的产品质量以外的保证，新能源开发公司A在合并财务报表层面不能于出售电站时确认电站出售收入；对于情形二，会议讨论初步形成倾向性意见，基于本案例的背景，新能源开发公司A在合并财务报表层面不能于出售电站时确认电站出售收入。

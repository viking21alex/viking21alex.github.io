案例3《会议纪要》–关于新租赁准则的分步执行衔接处理​

2019年第一次会议

【背景或案例】新租赁准则要求境内A+H股公司在2019年1月1日执行新租赁准则，而境内非上市公司则主要在2021年执行。在此情况下，A+H股公司对于持有境内非上市公司参股公司，在2019年难以完全掌握参股公司新租赁准则的财务影响情况下，只能以未来适用法先计提参股公司新租赁的财务影响，反映在2019年的报表内。在2021年，如参股公司届时选取部分追溯法执行新租赁准则，A+H股上市公司应如何调整参股公司租赁执行方式的差异？是否只能在2021年内当年消化与预估数据之间的差异，还是可以再次进行追溯调整？

会议分析：这是一个共性问题，准则分阶段实施都会遇到此问题。在分步执行背景下，可考虑结合重要性原则进行处理，即如果金额影响较小，建议投资方（联营股东）在其当期财务报表中处理；如果金额较大，则允许投资方按参股公司首次执行新租赁准则的影响追溯调整投资方的期初报表，但不调整比较期间的信息。

会议初步意见：建议允许企业按差异金额是否重大，分别于当期处理，或追溯调整期初。

案例2《会议纪要》–关于反向购买交易费用的会计处理问题

案例2《会议纪要》–关于反向购买交易费用的会计处理问题​

2017年第一次会议

证监会于2016年9月发布《关于修改〈上市公司重大资产重组管理办法〉的决定》，取消了重组上市(俗称“借壳上市”)的配套融资安排。“借壳上市”属于企业会计准则体系中所规范的反向购买的情况。重组新政前，由于借壳交易同时配套募集资金，实务中，往往较大比例的中介费用是配套募集资金的承销费，因此冲减了发行收入。然而，重组新政取消配套融资安排之后，在“借壳上市”交易中，上市公司不再允许配套募集资金。此时，独立财务顾问费、会计师审计费、律师费以及评估费等交易费用是否属于与发行金融工具（股票）直接相关的费用，应当如何进行相应会计处理，在实务中存在一定的争议。

会议分析：该问题涉及企业合并准则。根据准则规定，购买方为企业合并发生的审计、法律服务、评估咨询等中介费用以及其他相关管理费用，应当于发生时计入当期损益，购买方作为合并对价发行的权益性工具或债务性工具的交易费用，应当计入该工具的初始确认金额。在反向购买交易中，如果无法区分哪些是与会计上的被购买业务相关的，哪些是与股份发行相关的，可以考虑简化处理。

会议初步意见：现行准则和解释等已体现了处理原则，企业应当据此进行会计处理。

案例3《会议纪要》–关于反向购买中业绩对赌的会计处理

案例3《会议纪要》–关于反向购买中业绩对赌的会计处理​

2023年第一次会议

   【背景或案例】企业会计准则中规定了非同一控制下企业合并与同一控制下企业合并的或有对价的会计处理原则。前者按照公允价值计量且其变动计入当期损益（不构成权益工具的情况下，下同）；后者按照或有事项的原则，且后续变动计入权益。但对于反向购买交易中，与标的企业（会计上的购买方）的业绩对赌，实务中应如何会计处理不明确。如A上市公司向P公司发行股份，收购其全资子公司B公司，同时针对B公司之后三年的净利润进行了业绩对赌，P公司将基于净利润的差额对上市公司进行补偿，返还上市公司的股份，该补偿安排不满足权益工具的定义。A和B均构成业务，该交易属于反向购买。问题1：反向购买中，与标的企业（会计上的购买方）的业绩对赌，应该按照公允价值进行初始和后续计量，还是按照或有事项准则进行确认和计量？问题 2：反向购买中，针对标的企业（会计上的购买方）的业绩对赌，后续计量与初始计量的差额应该计入损益还是权益？

会议分析，关于问题1，与会代表一致认为，反向购买属于非同一控制下企业合并，不属于同一控制下企业合并，因此，相关业绩对赌应该以公允价值进行初始和后续计量，不适用或有事项准则。关于问题2，多数与会代表认为应计入损益，其中部分与会代表认为，应该参考或遵循非同一控制下企业合并的或有对价会计处理原则，部分与会代表认为应该按照金融工具准则计量，分类为公允价值计量且其变动计入当期损益的金融资产；同时，少数与会代表认为应计入权益，因为反向购买交易完成后，上市公司合并报表层面的报告主体实质上是标的企业（会计上的购买方），因而针对标的企业的业绩对赌，实质上是报告主体与其控股股东之间就报告主体自身业绩进行的对赌，属于权益性交易，后续变动应计入权益。

会议初步意见，该问题具有一定的代表性，会议讨论初步形成倾向性意见，关于本案例中反向购买的业绩对赌，企业应当结合并购意愿、商业实质等，按照现行企业会计准则的有关规定进行会计处理。

案例8《会议纪要》–关于对赌条款形成的金融负债的后续计量​

2021年第二次会议

【背景或案例】拟上市公司在引入财务投资者时，往往约定特殊权益安排，即在发生某些约定事件时会触发股份回购、反稀释、清算优先等条款，俗称对赌条款。2020年企业会计准则实施技术联络小组第一次会议中曾讨论过，从发行人角度看，当发行人不能无条件避免交付现金或其他金融资产以履行回购义务时，应当将其分类为金融负债。实务中，上述对赌条款可能在增资协议之外同时签署的补充协议中作出约定（情形一），也可能全部写入增资协议和公司章程（情形二）。两种情形下对赌条款形成的金融负债的后续计量是否不同？

会议分析：该问题具有普遍性和代表性，多数与会代表认为，在我国现行法律体系下，回购、反稀释、清算优先等特殊权益安排导致附带特殊权益安排的普通股与一般普通股不同，而且仅因特殊权益是否在单独补充协议中约定而做出不同的会计处理可能会诱导市场构建交易，不建议对上述两种情形区别对待。即，对于上述增资，均应当分类为金融负债，其金额等于回购所需支付金额的现值（按照摊余成本进行后续计量），反稀释条款构成一项单独的衍生金融负债。对于企业按照有关规定已经履行了注册登记等增资手续的，企业应当根据收到认股款确认实收资本和资本公积（资本溢价）；同时，就回购义务确认负债（作收购库存股处理）。少数与会代表认为应当根据实际情况判断，对于情形一赞同多数代表的意见；对于情形二，要进一步分析该特殊权益是与投资人身份挂钩还是与普通股股份挂钩，若为前者，与情形一处理一致，若为后者，其经济实质与境外法律环境下发行的可赎回可转换优先股类似，多数与会人员认为，该负债属于主合同为金融负债，嵌入回售权、转股权及优先清算权的混合工具，因此，可以将嵌入衍生工具单独拆分并对主合同的金融负债采用摊余成本计量，也可以选择将整个混合工具指定为以公允价值计量且其变动计入当期损益的金融负债。

会议初步意见：该问题具有普遍性和代表性，拟通过应用案例形式进一步加强实务指导。

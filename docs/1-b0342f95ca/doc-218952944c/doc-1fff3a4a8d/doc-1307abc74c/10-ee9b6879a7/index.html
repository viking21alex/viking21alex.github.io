案例10《会议纪要》–关于数字人民币的会计处理

案例10《会议纪要》–关于数字人民币的会计处理​

2021年第三次会议

【背景或案例】2019年底，我国开始数字人民币试点。中国人民银行数字货币研究所数据显示，截至2021年10月22日已经开立数字人民币个人钱包1.4亿个，企业钱包1000万个，累计交易笔数达到1.5亿笔，交易额接近620亿元。目前，有155万商户支持数字人民币钱包，涵盖公共事业、餐饮服务、交通出行、购物和政务等各个方面。随着数字人民币的不断普及，数字人民币的具体会计处理需要进一步明确，比如数字人民币应当在哪个会计科目核算，在资产负债表哪个报表项目列报，是否符合现金流量表中现金的概念。

会议分析，该问题具有普遍性和代表性。数字人民币是由中国人民银行发行的数字形式的法定货币，由指定运营机构参与运营并向公众兑换，以广义账户体系为基础，支持银行账户松耦合功能，与纸钞硬币等价，具有价值特征和法偿性，支持可控匿名。与会代表一致认为，数字人民币符合货币资金的定义，应当在资产负债表中列报为货币资金，在判断是否符合现金流量表中现金的概念时，应与其他类型的货币资金判断标准一致，在现行企业会计准则体系下能够解决。在数字人民币核算的具体会计科目方面，部分与会代表建议在“其他货币资金”科目下设置二级明细科目核算；部分与会代表建议在“库存现金”科目设置二级明细科目核算，或将“库存现金”改为“现金”且下设“库存现金”和“数字货币”两个明细科目；部分与会代表建议新增“数字货币”一级科目核算。

会议初步意见，该问题具有普遍性和代表性，对于数字人民币的列报项目和判断是否符合现金流量表现金的概念应当遵循现行企业会计准则体系的相关规定，拟通过会议纪要形式统一理解认识。对于数字人民币核算的具体会计科目，将结合实务发展决定是否有必要作进一步规范。

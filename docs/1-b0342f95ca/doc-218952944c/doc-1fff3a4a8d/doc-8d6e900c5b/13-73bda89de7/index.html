案例13《会议纪要》–关于长期股权投资和“三无”投资核算单元的认定​

2021年第三次会议

【案例或背景】关于投资方取得的对被投资方的股权能否区分不同的计量单元采取不同的会计处理，具体涉及如下情形：（1）如果同一集团内不同主体分别购买了同一上市公司的股票且均不构成重大影响，各主体分别将相应投资划分为“其他权益工具投资”和“交易性金融资产”，在集团合并层面是否需要重新统一进行会计处理？（2）如果一家企业在不同时间分别购买了同一上市公司的股票但不构成重大影响，是否可以分别将相应投资划分为“其他权益工具投资”和“交易性金融资产”？（3）如果同一集团内不同主体分别持有同一被投资单位的股权，并分别适用长期股权投资准则和金融工具准则，如某一主体按照权益法或成本法核算，另一主体按照金融工具准则核算，在集团合并报表层面是否需要统一调整为按长期股权投资准则核算？（4）如果企业持有一项具有重大影响的股权投资并按权益法核算，后续取得被投资单位对全体股东进行配股分配的股权，公司取得的该部分配股股权可否不计入权益法核算的长期股权投资而另行计入“交易性金融资产”？

会议分析，该问题具有普遍性和代表性。多数与会代表认为，在合并报表层面应当分别如下情况考虑：一是当对同一被投资单位的股权投资整体不构成控制、共同控制或重大影响（即不适用长期股权投资准则）时，同一集团内不同主体、或者同一主体内不同部门所持有的对同一被投资单位的权益工具投资，按照金融工具准则的规定，可基于“单项”权益工具投资的持有目的分别划分为“其他权益工具投资”和“交易性金融资产”。二是当在合并报表层面对同一被投资单位的股权投资整体具有控制、共同控制或重大影响时，其中：（1）对于联营或合营企业投资，在合并报表层面需统一考虑直接加间接的持股比例进行权益法核算，此时对同一家被投资单位的权益工具投资，不能分别作为长期股权投资和金融资产。这种情况下，除非投资方对联营企业的权益性投资，其中一部分通过风险投资机构、共同基金、信托公司或包括投连险基金在内的类似主体间接持有的，则投资方可以按照金融工具准则的有关规定，对间接持有的该部分投资选择按照金融工具确认和计量准则的规定以公允价值计量且其变动计入损益，并对其余部分采用权益法核算。（2）对于子公司投资，合并报表层面需统一考虑直接加间接的持股比例进行合并报表的编制。

会议初步意见，该问题具有普遍性和代表性。现行《企业会计准则第2号——长期股权投资》、《企业会计准则第22号——金融工具确认和计量》等已有明确规定，企业应当按照相关准则进行会计处理。对于所持有的不具有控制、共同控制或重大影响的权益工具投资，可基于“单项”进行分类；对于从集团合并层面整体来看具有控制、共同控制或重大影响的权益工具投资，应在合并报表层面统一按权益法核算（除非是通过风险投资机构、共同基金、信托公司或包括投连险基金在内的类似主体间接持有的部分）联营企业或合营企业投资，按合并报表编制原则核算子公司投资。拟通过会议纪要形式统一理解认识。

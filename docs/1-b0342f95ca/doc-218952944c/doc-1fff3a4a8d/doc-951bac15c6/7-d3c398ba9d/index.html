案例7《会议纪要》–关于政府拨付的研发资金属于与资产相关还是与收益相关的政府补助

案例7《会议纪要》–关于政府拨付的研发资金属于与资产相关还是与收益相关的政府补助​

2022年第四次会议

【背景或案例】政府向A公司拨付研发资金，用于开发某种射频芯片项目，考核指标包括申请发明专利不少于10项等。项目总经费500万元，其中中央财政拨付400万元，A公司自筹100万元。项目总经费支出500万元中，包括购置设备、购买非专利技术、材料费、测试化验加工费、差旅费、劳务费以及其他费用，其中购买非专利技术、购置设备以及测试化验加工费的占比较大。问题1：若政府拨付的经费400万中，明确规定用于购置非专利技术、设备以及测试化验加工费等不同类型支出所对应的具体金额，则政府拨付的400万元属于与资产相关还是与收益相关的政府补助？问题2：若政府拨付的经费400万中，并没有明确规定用于购置非专利技术、设备以及测试化验加工费等不同类型支出所对应的具体金额，则政府拨付的400万元属于与资产相关还是与收益相关的政府补助？

会议分析，关于问题1：多数与会代表认为，如果政府下达的任务书或者企业上报并经批准的预算中明确了设备、非专利技术、其他研发费用等项目在中央财政资金预算数中的对应金额，则根据预算数将与设备、非专利技术相关的政府补助归类为与资产相关的政府补助；与其他研发费用（如测试化验加工费）相关的政府补助，若相关的研发费用根据研发支出资本化的条件，形成无形资产的，则对应的中央财政资金预算数作为与资产相关的政府补助，未形成无形资产的，则作为与收益相关的政府补助。关于问题2：多数与会代表认为，如果政府下达的任务书或者企业上报并经批准的预算中并未明确具体项目支出金额，企业首先应考虑采用适当的方法根据项目总经费支出中形成固定资产、无形资产的金额和费用化的金额的相对比例对政府补助进行拆分，分别归类为与资产相关的政府补助和与收益相关的政府补助；不能简单地以补助文件中未明确区分或者以“难以区分”为由，将综合性政府补助直接整体归类为与收益相关的政府补助。

会议初步意见，该问题具有一定的普遍性和代表性，会议讨论形成倾向性意见，对于综合性政府补助，企业应按照《企业会计准则第16号——政府补助》有关规定，采用合理的方法区分为与资产相关的政府补助和与收益的政府补助，不能简单地以补助文件中未明确区分或者以“难以区分”为由，直接整体归类为与收益相关的政府补助。

【容诚】问题21、存在预付租金的情况下，与租赁负债相关的递延所得税影响的会计处理​

（1）问题背景

2021年12月，承租人甲公司与出租人乙公司签订了为期4年的商铺租赁合同，至2022年1月1日起租。每年的租赁付款额为12万元（不含税），在前一年的年末支付。甲公司增量借款利率为5%，假定没有初始直接费用。甲公司于2021年底支付乙公司2022年度租金12万元。在租赁期开始日（2022年1月1日），甲公司按租赁付款额的现值所确认的租赁负债为326,789.76元，使用权资产为446,789.76元（326,789.76+120,000.00）。甲公司在租赁期内按照直线法对使用权资产计提折旧。假定按照适用税法规定，该交易属于税法上的经营租赁，每年末预付的租金允许在下一年度进行税前抵扣，甲公司适用的所得税税率为25%。假设甲公司未来期间能够取得足够的应纳税所得额用以抵扣可抵扣暂时性差异，不考虑其他因素。

问题：甲公司如何按照《企业会计准则解释第16号》的要求确认与使用权资产和租赁负债相关的递延所得税影响，以及未来期间转回的处理？

（2）问题分析

《企业会计准则解释第16号》：“一、关于单项交易产生的资产和负债相关的递延所得税

不适用初始确认豁免的会计处理 该问题主要涉及《企业会计准则第18号——所得税》等准则。

（一）相关会计处理。

对于不是企业合并、交易发生时既不影响会计利润也不影响应纳税所得额（或可抵扣亏损）、且初始确认的资产和负债导致产生等额应纳税暂时性差异和可抵扣暂时性差异的单项交易（包括承租人在租赁期开始日初始确认租赁负债并计入使用权资产的租赁交易，以及因固定资产等存在弃置义务而确认预计负债并计入相关资产成本的交易等，以下简称适用本解释的单项交易），不适用《企业会计准则第 18 号 ——所得税》第十一条（二）、第十三条关于豁免初始确认递延所得税负债和递延所得税资产的规定。企业对该交易因资产和负债的初始确认所产生的应纳税暂时性差异和可抵扣暂时性差异，应当根据《企业会计准则第18号——所得税》等有关规定，在交易发生时分别确认相应的递延所得税负债和递延所得税资产。”

《企业会计准则第18号——所得税》：“第五条 资产的计税基础，是指企业收回资产账面价值过程中，计算应纳税所得额时按照税法规定可以自应税经济利益中抵扣的金额。

第六条 负债的计税基础，是指负债的账面价值减去未来期间计算应纳税所得额时按照税法规定可予抵扣的金额。”

（一）初始确认

在租赁期开始日，甲公司租赁负债的账面价值为326,789.76元，计税基础（即账面价值减去未来期间计算应纳税所得额时按照税法规定可予抵扣的金额）为 0，产生可抵扣暂时性差异326,789.76元；甲公司使用权资产的账面价值为446,789.76元，计税基础（即收回资产账面价值过程中计算应纳税所得额时按照税法规定可以自应税经济利益中抵扣的金额）为120,000.00元（即预付的部分可以抵扣），产生应纳税暂时性差异326,789.76元。综上，租赁负债和使用权资产的暂时性差异金额相同，确认等额的递延所得税资产和递延所得税负债各81,697.44元（326,789.76×25%=81,697.44）。

另一种理解是使用权资产的计税基础也是0，按照《企业会计准则解释第16号》只确认与租赁负债的可抵扣暂时性差异等额的部分，其结果也是81,697.44元。

事务所倾向于前一种理解。

（二）后续与使用权资产相关的暂时性差异的转回处理

关于使用权资产相关的暂时性差异后续期间的转回，实务处理相对比较统一，即按直线法在折旧年限内平均分摊。本案例即每年按使用权资产中确认递延所得税负债的暂时性差异部分326,789.76元在4年内按直线法分摊确定每年暂时性差异的转为金额为81,697.44元，相应每年确认递延所得税负债减少的金额为20,424.36元。

（三）后续与租赁负债相关的暂时性差异的转回处理

实务中存在两种不同的观点。

由于存在预付的情况，按这一观点计算将导致递延所得税资产与递延所得税负债的转回情况不匹配。极端情况下，如果在某一年度一次性付款，会导致租赁负债对应的递延所得税资产在某一期间大额转回，而其他期间则没有或只有少量的发生额（利息部分）。反之，如果前面各期拖延不付款，到最后一期才一次性支付，则会导致递延所得税资产在前几期只有少量的转回（利息部分），而到最后一期才发生大额转回，这显然是不合理的。

观点二：相关递延所得税资产的后续转回情况，应当结合税法可抵扣的情况具体分析。

事务所倾向于观点二，具体分析如下：

甲公司2021年预付的12万元，在2022年度可以税前抵扣，以此顺延，2022年末支付租金导致的租赁负债的减少，是在2023年才能抵扣。因此，租赁负债因预付导致的减少不应作为当期暂时性差异的转回，而是在其可抵扣的年度才转回。也就是说，租赁负债增减变动中，确认利息费用导致的变动可直接确认为当期可抵扣暂时性差异的增加，但支付租金导致的变动需具体辨认，在其可抵扣的年度作为租赁负债暂时性差异的转回。本例中，2022年末支付的租金作为2023年暂时性差异的转回，则2022年度暂时性差异转回的金额仅有未确认融资费用变动的部分（实际是增加了暂时性差异），即利息变动的部分。2023年转回的金额为当年可抵扣的租金部分12万元与利息变动部分的净额，以此类推。则每年转回情况见下表：

关于使用权资产和租赁负债初始确认的递延所得税资产和递延所得税负债未来转回的处理，实务中存在较多不同理解，亟需讨论一个相对统一的原则和处理方法。

讨论情况：

与会者普遍认为，按照税法规定，经营租赁相关预付租金允许在下一年度进行税前抵扣。实务中，对于预付租金相关递延所得税的确认有三种处理方法，分别是作为使用权资产的计税基础、作为租赁负债的计税基础或单独作为一项计税基础（不与使用权资产和租赁负债挂钩）。三种方法对所得税费用的影响是一致的，并且当递延所得税资产与递延所得税负债净额抵销列报时，三种方法对报表列报项目也没有影响，只是在预付租金情况下，暂时性差异后续转回金额不一定等于使用权资产或租赁负债账面价值变动金额。

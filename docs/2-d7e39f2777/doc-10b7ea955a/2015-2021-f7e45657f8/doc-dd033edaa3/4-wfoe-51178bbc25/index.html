案例4 红筹架构拆除过程中高溢价收购WFOE公司股权的会计处理​

2018年8月3日会计问题讨论情况通报

【背景或案例】红筹架构搭建情况：

A公司为境内经营实体（拟IPO企业，VIE公司），自然人甲为公司高管，受境外C公司指派成为A公司名义股东（甲并未出资），实际控制人为境外自然人乙。实际控制人采用常见的协议控制模式，直接持有境外C公司的股权，境外C公司在境内设立外商独资企业B公司（即所指的WFOE公司），B公司通过一系列协议控制境内经营实体A公司。在红筹架构拆除前，B公司有研发人员、知识产权，并从事经营活动，但业务均发生在与A公司之间。

红筹架构图：

[资源缺失：media/2e3cf3acefc7c35063544310b3e7bc5b.png]

红筹架构拆除情况：

2015年末，自然人甲、A公司与自然人乙及其控制的境外C公司签署了如下一系列协议：

1.自然人甲向境外自然人乙支付资本补偿金3000万元，各方签署《终止协议》，解除A公司的股权质押，自然人甲取得对A公司的控制权。

A公司与B公司的股东境外C公司签订股权转让协议，收购B公司的100％股权，股权受让价格为1亿元，A公司向C公司支付远高于B公司股权公允价值的股权受让款，将B公司变为A公司的全资子公司。

3.自然人甲负责为A公司筹集资金，通过引入新投资人D公司，取得增资款9000万元。

A公司并购日为2015年12月31日，并购日A公司的净资产6000万元，B公司的净资产为-100万元，B公司可辨认净资产的公允价值为2000万元（主要资产为评估增值的知识产权，假设该金额与B公司股权的公允价值相同）。收购完成后，B公司在2017年被注销，B公司的研发人员和知识产权转移到A公司。

拆除红筹架构后控制关系图：

[资源缺失：media/7b198506c5a61f09c556b8bb09ff1052.png]

问题：对于A公司收购B公司100%股权所支付的股权受让款，在A公司个别报表和合并报表中应如何进行会计处理？

实务中的不同观点：

观点一：

对A公司支付的高于B公司股权公允价值的股权转让款，应当按照权益性交易的原则进行处理，不确认商誉或当期损益。

自然人甲向境外自然人乙支付资本补偿金与A公司收购B公司100%股权属于一揽子交易，最终目的就是使自然人甲获得A公司的控制权。A公司为收购B公司100%股权，向境外C公司支付的款项包含两部分构成，即B公司股权的受让款和A公司原实际控制人自然人乙的投资退出款，B公司股权受让款可以以B公司股权的公允价值作为基础，将支付的款项中高于B公司股权公允价值的部分认定为支付的投资退出款冲减资本公积，资本公积不足冲减的，冲减留存收益。

A公司在编制合并财务报表时，按照非同一控制下企业合并进行会计处理。

观点二：

A公司（VIE）收购B公司（WFOE）的交易中不涉及权益性交易，全部作为非同一下控制下企业合并进行会计处理。

拟上市主体VIE和WFOE公司在VIE架构拆除前后实际控制人发生了变化，拆除前的实际控制人为自然人乙，拆除后的实际控制人为自然人甲，根据企业合并准则的相关规定，不符合同一控制下企业合并。如果控股股东(包括单一控制方以及控制方组合)发生变化，应采用购买法，以公允价值核算WFOE的资产、负债，且不重溯历史业绩。A公司个别报表会计处理：将支付对价全部确认为股权对价，合并报表中A公司收购对价为1亿元，B公司可辨认净资产为2000万元，应确认8000万元商誉。

观点三：

实务中，绝大多数拆VIE、拆红筹的重组交易中，B公司本身往往并不构成一项单独的业务，通常历史上B公司从事的业务与A公司相互交织，不可割裂，B公司与A公司的结合才能构成一项完整的业务。在这种情况下，实务中可能会存在两种不同的意见。

第一种意见认为，由于A公司与B公司构成一项完整的业务，整个业务没有发生变化，即使控制方发生变化，仍属于业务的法律形式的重组，仍应沿用业务原来的账面价值，即与原B公司合并报表保持一致，且重述历史业绩。A公司的个别报表中，可按B公司股权的公允价值2000万作为长期股权投资成本，与股权转让款的差额作为权益性交易处理。

第二种意见认为，由于控制方发生了变更，因此交易总体上不属于权益性交易，其实质更接近于非同一控制下的企业合并，应该采用公允价值来进行计量。从A公司的层面看，其收购B公司应该采用公允价值来计量其取得的各项可辨认资产和负债，但是由于A公司和B公司合在一起才构成一项完整的业务，如果只调整B公司的资产为公允价值而A仍用账面价值计价，财务报表整体从结果导向而言并不合理，因此，可以考虑从变更后的控制方的角度来进行公允价值的下推会计，即A+B业务整体均以公允价值计量，且不重述历史业绩。

讨论情况：经讨论，在A公司合并层面，与会者一致认同观点三中的第一种意见。对于VIE架构，由于A和B整体才构成完整的业务，甲收购A，与A收购B实质为一揽子交易，一系列架构拆除安排的实质为A和B业务的股东从境外的自然人乙和公司C变为境内自然人甲和投资人D，境内主要经营实体A和B的资产和业务并未发生变化，应将A和B看做一个整体，其控股股东改变，并不会改变其计量基础，因此在A公司合并报表层面， A和B分别采用不同的计量基础不符合交易实质，仍应沿用业务原来的账面价值， A收购B所支付的对价超出B公司账面净资产部分实质为A代替其股东甲支付给原股东乙的投资退出款，应作为权益性交易处理。

口头通知：

  1. 权益法核算被投资单位除净损益、其他综合收益以及利润分配以外的其他所有者权益变动，投资方的 “资本公积—其他资本公积”可以出现负数；该负数在相对应的长期股权投资发生减值时，不应转出至损益。

  2. 以买卖上市公司壳资源为目的的收购，如果在购买日上市公司还保留有业务，则会构成非同一控制下企业合并，由此形成的巨额商誉，后续发生减值只能计入损益。

  3. 有限责任公司改制为股份有限公司时，个别报表中的未分配利润会全额结转至股本/资本公积。如果存在之前集团内交易形成的未实现利润，合并报表层面的未分配利润会低于个别报表层面的未分配利润。此种情况下，合并报表层面结转至股本/资本公积的未分配利润金额仍应与个别报表层面相等，即使结转之后合并报表层面的未分配利润会出现负数。

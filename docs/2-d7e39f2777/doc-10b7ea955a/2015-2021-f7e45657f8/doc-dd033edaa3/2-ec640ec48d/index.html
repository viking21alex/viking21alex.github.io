案例2 分步购买不构成业务的公司的会计处理​

2018年8月3日会计问题讨论情况通报

【背景或案例】

1、甲公司（房地产开发公司）购买A公司60%的股权

乙公司持有A公司100%股权，A公司为一个房地产项目公司，账面仅有一块土地。甲公司与乙公司为非关联方。2015年1月1日，甲公司购买了A公司60%股权，由于A公司仅有一块未开发的土地，无雇员，且A公司尚未开始项目开发的主要活动和生产产出的计划，故本次收购不构成业务合并。购买日，土地的账面价值500万元，评估值（公允价值）为1000万元，经双方协商，甲公司以720万元（溢价120万元，溢价率20%）的价格收购了A公司60%股权，并将A公司纳入合并报表范围。

2、甲公司进一步购买A公司剩余40%股权

【情形一】2015年7月1日，甲公司进一步购买了A公司40%股权。假设，这半年中A公司仍未开始项目开发的主要活动和生产产出的计划，仍然不构成业务，但公允价值升至1200万元；两次交易也不构成一揽子交易。甲公司进一步购买40%股权支付的对价为576万元（溢价96万元，溢价率仍为20%）。

【情形二】2017年1月1日，甲公司进一步购买A公司40%股权。此时，房屋基本已经建成，发生的建设成本为3000万元，估值为5000万元。甲公司进一步购买40%股权的对价为2000万元。

A公司不构成业务的情况下，溢价购买A公司股权的商业合理性。

（1）相关交易为符合市场交易规则的收购双方自愿达成的交易价格；

（2）土地获取通常会受到当地政府的政策干预，例如当地政府会鼓励当地开发商获得待开发地块的土地使用权，而其他地区的开发商往往会受到限制获得竞拍，因此开发商在拓展地域限制，获得新的地块开发权，通常只有与其他地区已获得土地开发权、但资金紧张的开发商进行合作，由于地块取得一段时间后，土地资源的稀缺性，经评估的地块价值通常比标的公司原竟得土地的成交价格要高（溢价率要看地块所处位置、容积率、市政配套等各项指标），新股东溢价购买地块也是出于对项目未来的预期收益，愿意承担高出原土地成交价格的成本，同时地块的溢价也存在对标的公司控制权的溢价因素。

目前，房地产开发商跨地域获得地块，且控制标的公司，溢价收购标的公司所持有的地块开发权普遍存在，有其商业合理性。

问题：

1、2015年1月1日，甲公司第一次购买A公司60%股权时，甲公司合并报表中，土地的初始确认成本是多少？

2、在情形一和情形二下，甲公司进一步收购A公司40%股权时，在合并报表中如何进行会计处理？按照购买少数股权处理？还是仍然作为购买资产处理，将溢价部分增加土地成本？

实务中的不同观点：

问题1：资产初始确认成本

观点一：土地初始确认成本应为所支付对价（720万股权转让款）与40%NCI的价值之和。其中，如果40% NCI可以享有对A公司的现时所有者权益，并且可以在A公司清算时享有其净资产的相应份额，则可以选择以其所享有的净资产公允价值份额（即1000万*40%）或其本身公允价值计量；否则只能按照其本身的公允价值计量。对少数股东权益公允价值的评估应基于少数股东享有的权利（包括现时所有者权益、若发生清算享有净资产份额等），并充分考虑土地评估值1000万的估值方法和基础，以及甲公司以20%溢价率收购的原因。

观点二：A公司土地的初始确认成本为甲公司支付的对价720万元和少数股东所享有的净资产公允价值份额（即1000万*40%，400万元）。

观点三：根据资产初始计量的一般原则，应采用支付对价的公允价值作为资产的初始入账成本，即按照720万元对价和60%比例折算的1200万元，确认少数股东权益480万元。同时，甲公司需要重新评估购入土地的成本是否存在减值，如果存在减值，则应当计提减值准备，计入当期损益，少数股东也应分担相应的减值损失。

观点四：不构成业务时，不存在合并报表、分步购买及购买少数股权的会计处理，资产按公允价值入账，实际对价高于公允价值的应作为费用处理。

观点五：认为将购买A公司识别为购买一项业务，更符合交易的商业实质及企业合并准则的规定。

问题2：进一步收购A公司40%股权

观点一：无论在收购40%股权的时候，A公司是否构成业务，都应按照收购少数股东权益进行会计处理。

观点二：在A公司不构成业务的情况下，进一步收购A公司40%股权作为购买资产处理，支付对价超过持续计量的少数股东权益部分增加资产成本；若构成业务，则按照购买少数股权处理。

讨论情况：

问题1：资产初始确认成本

与会者普遍认为，合并财务报表准则规定，合并范围应当以控制为基础确定，将子公司所有者权益中不属于母公司的份额作为少数股东权益处理，不区分子公司是否构成业务，企业合并准则与合并报表准则中对于子公司的并表存在范围上的差异。本案例中，初次购买日，首先需要判断标的公司是否构成业务，以及在不构成业务的情况下，溢价购买资产是否具有商业合理性，资产公允价值的估计是否合理，并提供充分恰当的证据。

经讨论，在标的公司不构成业务的情况下，如果购买方购买标的公司60%的股权后，已经控制该公司，则购买方合并报表应按照标的公司土地公允价值入账（包括购买方已支付对价和少数股东权益的价值）。本案例中，如果720万元反映60%土地的公允价值，则资产的初始入账成本应为1,200万元（720万元按照60%比例折算），同时确认少数股东权益480万元；但如果720万元中除相应资产的公允价值外还包含控制权溢价等其他因素，那么资产的初始入账成本应为已支付720万元对价和少数股东所享有的净资产公允价值份额。同时，当资产的初始确认金额高于其公允价值的情况下，需考虑资产减值。

问题2：进一步收购A公司40%股权

经讨论，在购买方控制标的公司后，即资产在合并报表中初始确认后，控制权不变的情况下，其计量基础应保持不变，与进一步收购40%股权是否构成业务无关。后续进一步收购标的公司40%股权时，无论是否构成业务，都应该按照收购少数股权进行核算，而不能调整资产的账面价值。

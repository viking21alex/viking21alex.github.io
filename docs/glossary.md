# 术语表

## 半导体与制造

| 术语 | 英文 | 说明 |
|------|------|------|
| 晶圆 | Wafer | 半导体制造的基础衬底，通常为硅晶圆 |
| 制程节点 | Process Node | 衡量芯片制造工艺精密度的指标，如 7nm、5nm、3nm |
| 光刻 | Lithography | 将电路图案转移到晶圆上的关键工艺 |
| 刻蚀 | Etching | 从晶圆表面去除材料以形成电路结构的工艺 |
| 先进封装 | Advanced Packaging | 超越传统引线键合的芯片封装技术，如 CoWoS、Chiplet |
| HBM | High Bandwidth Memory | 高带宽内存，通过 3D 堆叠实现高数据吞吐量 |
| Chiplet | 小芯粒 | 将大芯片拆分为多个小芯粒的模块化设计方法 |

## 芯片设计

| 术语 | 英文 | 说明 |
|------|------|------|
| GPU | Graphics Processing Unit | 图形处理器，AI 训练的主力芯片 |
| ASIC | Application-Specific Integrated Circuit | 针对特定应用定制的芯片 |
| NPU | Neural Processing Unit | 神经网络处理器，专为 AI 推理优化 |
| FPGA | Field-Programmable Gate Array | 可编程逻辑门阵列，灵活但效率较低 |
| IP核 | IP Core | 可复用的芯片设计模块，如 ARM CPU 架构、PCIe 接口 |
| EDA | Electronic Design Automation | 芯片设计自动化工具 |

## AI 软件与模型

| 术语 | 英文 | 说明 |
|------|------|------|
| LLM | Large Language Model | 大语言模型，如 GPT、Claude |
| 多模态 | Multimodal | 能处理文本、图像、音频等多种输入类型的模型 |
| 推理 | Inference | 已训练好的模型对新数据做出预测的过程 |
| 训练 | Training | 用大量数据调整模型参数的过程 |
| 量化 | Quantization | 降低模型精度以减小模型大小和加速推理 |
| MLOps | Machine Learning Operations | 模型开发、部署和运维的工程实践 |

## 算力基础设施

| 术语 | 英文 | 说明 |
|------|------|------|
| 智算中心 | Intelligent Computing Center | 专门为 AI 计算优化的数据中心 |
| 液冷 | Liquid Cooling | 使用液体冷却服务器，比风冷效率更高 |
| 算力网络 | Computing Power Network | 跨地域的算力调度和聚合系统 |
| TCO | Total Cost of Ownership | 总拥有成本，含采购、运维、能源成本 |
| PUE | Power Usage Effectiveness | 数据中心能效指标，越接近 1 越高效 |

## 行业应用

|------|------|------|
| 具身智能 | Embodied AI | 将 AI 模型与物理世界交互的智能体 |

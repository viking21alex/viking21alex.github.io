# 项目概述与使用指南

## 这是什么

AI 全产业链知识库是一个面向研究者、从业者和学习者的结构化知识系统。它把 AI 产业从底层硬件到行业应用的完整链条拆解为七个层级，每个层级包含若干"卡口"（Bottleneck）——即产业链中关键技术、产品或环节。

## 为什么叫"卡口"

"卡口"（Bottleneck）指的是产业链中至关重要、且可能因技术壁垒、产能限制或地缘因素变得紧张的关键环节。识别和理解卡口，是理解 AI 产业链运作逻辑的核心。

## 三模块设计

### 1. 产业地图（Industry Map）
七层纵向产业链总览。展示各层级之间的上下游依赖关系，以及每个层级内的核心卡口分布。适合建立全局认知。

### 2. 卡口库（Bottleneck Library）
每个卡口的深度文档，包含：
- 技术概述与关键指标
- 竞品格局与市场格局
- 国产化状态与差距分析
- 关联公司索引

### 3. 公司库（Company Index）
每家公司结构化档案，包含：
- 基本信息与公司概述
- 核心产品与技术路线
- 在产业链中的位置
- 竞品对比与公开财务数据

## 如何阅读

1. **快速入门**：打开 `dashboard.html`，先看产业地图建立全局框架
2. **深入研究**：进入感兴趣的层级，阅读该层的总览文档（`layers/0X-*.md`）
3. **专题深潜**：点击具体卡口，阅读卡口文档（`bottlenecks/`）
4. **交叉验证**：通过公司文档（`companies/`）了解实际竞争格局

## 数据范围

- 覆盖地区：中国、美国、欧洲、日本、韩国、中国台湾
- 公司覆盖：上市公司 + 关键未上市初创公司
- 数据更新：持续维护
